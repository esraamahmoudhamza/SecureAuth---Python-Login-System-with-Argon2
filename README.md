# SecureAuth - Python Login System with Argon2

SecureAuth is a modern desktop login system built with Python and CustomTkinter, implementing secure password hashing using Argon2.

## 🔐 Features

- Argon2 password hashing
- Secure authentication system
- Login attempt tracking
- Modern secure dashboard
- Change password functionality
- Logout system
- Clean UI design

## 🛠 Technologies

- Python 3.x
- CustomTkinter
- Argon2-cffi

## 📂 Project Structure

SecureAuth/
│── main.py
│── security.py
│── users.db

## 🚀 Installation

1. Clone the repository
2. Install dependencies:

pip install customtkinter argon2-cffi

3. Run the application:

python main.py

## 🔒 Security Note

Passwords are never stored in plain text.
All passwords are hashed using Argon2 before being stored.
